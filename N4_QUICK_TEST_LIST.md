## 快速验证命令

### 数据库查询验证

```bash
# 检查这30个词的数据完整性
sqlite3 "NichiDict/Resources/seed.sqlite" "
SELECT
    de.headword,
    de.reading_hiragana,
    de.jlpt_level,
    de.frequency_rank,
    COUNT(DISTINCT ws.id) as senses,
    COUNT(es.id) as examples
FROM dictionary_entries de
JOIN word_senses ws ON de.id = ws.entry_id
LEFT JOIN example_sentences es ON ws.id = es.sense_id
WHERE de.headword IN (
    'だから', 'そんな', 'はっきり', 'きっと', 'もし',
    '上がる', '下げる', '謝る', '挨拶', '手伝う',
    '心配', '世界', '事故', '予定', '国際',
    '子', '君', '僕', '内', '召し上がる'
)
GROUP BY de.id
ORDER BY de.frequency_rank;
"
```
